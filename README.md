# PRAGMA Pentagon
### A Five-Dimension Post-Editing Framework for Pragmatic Quality Assessment in LLM Translation

> ⚠️ Work in Progress — Framework under active development

## Overview
PRAGMA Pentagon is an original evaluation framework designed to help 
professional translators and post-editors identify pragmatic failures 
in LLM-generated translations that traditional quality metrics miss.

The framework grounds post-editing practice in classical pragmatic 
theory — moving beyond fluency and terminology to assess communicative 
appropriateness.

## The Five Dimensions
| # | Dimension | Core Question |
|---|-----------|---------------|
| 1 | Speech Act Appropriateness | Does it perform the same communicative function? |
| 2 | Register & Formality | Is the tone right for the relationship? |
| 3 | Implicature & Cultural Meaning | Are "between the lines" meanings preserved? |
| 4 | Politeness Strategies | Are face-saving mechanisms culturally appropriate? |
| 5 | Context & Situation | Does it fit the specific communicative situation? |
